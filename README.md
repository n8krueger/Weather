# Project Title

Console Weather App

## Description

Application that will return current and forecasted weather based on zipcode input by user. Application runs in the console and uses a text-based 
menu for users to make selections. Weather info is being retrieved via REST api from www.weatherapi.com.

## Getting Started

### Dependencies

* OpenJDK 11.0.2
* System Environment Variable "WeatherAPI_KEY"

### Installing

* Copy/Paste jar file from out/artifacts/ to desired directory

### Executing program

* In console, navigate to directory containing jar file
```
java -jar Weather.jar
```
